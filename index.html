ระบบสะสมแต้ม

คู่มือการใช้งานระบบสะสมแต้มสำหรับผู้ใช้งานทั่วไป

1

เข้าสู่ระบบ

เข้าสู่ระบบด้วยเลขบัตรประจำตัวประชาชน หากยังไม่มีบัญชีผู้ใช้ โปรดติดต่อผู้ดูแลระบบเพื่อขอสร้างบัญชีใหม่

2

รีเฟรชข้อมูล

อัปเดตข้อมูลล่าสุดในระบบโดยการกดปุ่มรีเฟรช เพื่อให้แน่ใจว่าข้อมูลที่แสดงเป็นข้อมูลปัจจุบัน รวมถึงแต้มสะสม และประวัติการทำกิจกรรม

3

ตรวจสอบ E-Stamp

ดูจำนวน E-Stamp ที่ได้รับจากการเข้าร่วมกิจกรรมต่างๆ ตรวจสอบสถานะการสะสม และดูรายละเอียดการได้รับ E-Stamp แต่ละครั้ง

4

QR code สำหรับสะสม Stamp

แสดง QR code ส่วนตัวของคุณให้เจ้าหน้าที่สแกนเพื่อเพิ่ม E-Stamp เมื่อเข้าร่วมกิจกรรม QR code นี้จะใช้ในการระบุตัวตนและเพิ่มแต้มในระบบ

5

ประวัติการได้รับ E-Stamp

ดูประวัติการได้รับ E-Stamp ทั้งหมด รวมถึงวันที่ เวลา กิจกรรมที่เข้าร่วม และจำนวน E-Stamp ที่ได้รับ

ระบบจองรอบฐานกิจกรรม

คู่มือการใช้งานระบบจองรอบฐานกิจกรรมสำหรับผู้ใช้งานทั่วไป

1

ดูข้อมูลรอบฐานกิจกรรม

ดูข้อมูลรอบฐานกิจกรรมที่เปิดให้เข้าร่วม ตรวจสอบรายละเอียดกิจกรรม วันที่ เวลา สถานที่ และจำนวนที่นั่งที่เหลือ

2

ตรวจสอบข้อมูลการจอง

ตรวจสอบประวัติการจองของคุณ ดูสถานะการจอง รายละเอียดกิจกรรมที่จองไว้ และข้อมูลการยืนยันการเข้าร่วมกิจกรรม โดยการกรอกเลขบัตรประจำตัวประชาชน

ระบบสะสมแต้ม (เจ้าหน้าที่)

คู่มือการใช้งานระบบสะสมแต้มสำหรับเจ้าหน้าที่

1

เข้าสู่ระบบ

ให้ทำการเข้าสู่ด้วย Email และ Password ที่เจ้าหน้าที่ส่วนกลางได้ให้ไว้ หากไม่สามารถเข้าสู่ระบบได้ ให้ติดต่อเจ้าหน้าที่ฝ่าย IT

2

บันทึก E-stamp

เจ้าหน้าที่สามารถบันทึก E-stamp ให้ผู้เข้าร่วมกิจกรรม ด้วยวิธี Scan QR Code หรือ กรอกเลขบัตรประจำตัวประชาชน จากนั้นระบบจะทำการบันทึกข้อมูล เมื่อสำเร็จ หรือเกิดข้อผิดพลาดระบบจะแสดงแจ้งเตือน

ระบบจองรอบฐานกิจกรรม (เจ้าหน้าที่)

คู่มือการใช้งานระบบจองรอบฐานกิจกรรมสำหรับเจ้าหน้าที่

1

เข้าสู่ระบบ

ให้ทำการเข้าสู่ด้วย Email และ Password ที่เจ้าหน้าที่ส่วนกลางได้ให้ไว้ หากไม่สามารถเข้าสู่ระบบได้ ให้ติดต่อเจ้าหน้าที่ฝ่าย IT

เลือกเมนูที่สามารถใช้งาน

เมื่อทำการเข้าสู่ระบบแล้วระบบจะทำการแสดงเมนูขึ้นมาให้ใช้งาน

2

จองรอบฐานกิจกรรม ค้นหาข้อมูล

ทำการกรอกเลขบัตรประจำตัวประชาชนของผู้เข้าร่วมกิจกรรมเพื่อค้นหาข้อมูลในการจองรอบฐานกิจกรรม

เลือกฐานกิจกรรม

ระบบจะแสดงรอบฐานกิจกรรมที่สามารถจองได้ ขึ้นมาให้เลือก

แจ้งเตือนยันยันการจอง

ระบบจะทำการแสดงแจ้งเตือนยืนยันการจอง ให้ตรวจสอบก่อนทำการจองรอบฐานกิจกรรม

แจ้งเตือนจองรอบฐานกิจกรรมสำเร็จ

ระบบจะทำการแสดงแจ้งเตือนเมื่อจองสำเร็จ

3

Check-in ค้นหาข้อมูล

ทำการกรอกเลขบัตรประจำตัวประชาชนของผู้เข้าร่วมกิจกรรมเพื่อค้นหาการจอง

Check-in เข้าร่วมกิจกรรม

เมื่อข้อมูลการจองขึ้นมา ให้ทำการกดปุ่ม ยืนยัน Check-in เพื่อ Check-in เข้าร่วมฐานกิจกรรม

แจ้งเตือน Check-in สำเร็จ

เมื่อ Check-in สำเร็จระบบจะทำการแจ้งเตือนให้เจ้าหน้าที่ทราบ

4

รายการจอง

อนุมัติหรือปฏิเสธการจองของผู้ใช้ ตรวจสอบเงื่อนไขการเข้าร่วม และจัดการสถานะการจอง

ระบบสร้างเกียรติบัตร (เจ้าหน้าที่)

คู่มือการใช้งานระบบสร้างเกียรติบัตรสำหรับเจ้าหน้าที่

1

สร้างเกียรติบัตร

เจ้าหน้าที่จะต้องกรอกข้อมูลเพื่อสร้างเกียรติบัตรให้กับผู้เข้าร่วมกิจกรรม โดยสามารถกรอกข้อมูลใหม่ หรือกรอกเลขบัตรประชาชนเพื่อดึงข้อมูลและนำมาสร้างเกียรติบัตรอิเล็กทรอนิกส์

ตัวอย่างเกียรติบัตร

เมื่อเกียรติบัตรถูกสร้าง ระบบจะแสดงตัวอย่างเกียรติบัตรให้เจ้าหน้าที่เห็น

2

ตารางข้อมูลเกียรติบัตร

ตาารางแสดงข้อมูลเกียรติบัตร โดยสามารถจัดการเกียรติบัตรได้

ส่งเกียรติบัตรอีกครั้ง

หากต้องการส่งเกียรติบัตรอีกครั้ง สามารถกดปุ่มส่งอีเมลเพื่อส่งเกียรติบัตรอีกครั้งได้

3

ลบข้อมูลเกียรติบัตร

ลบข้อมูลเกียรติบัตรหากชื่อ-นามสกุลผิดพลาด

ระบบตรวจสอบเกียรติบัตร (เจ้าหน้าที่)

คู่มือการใช้งานระบบตรวจสอบเกียรติบัตรสำหรับเจ้าหน้าที่

1

ตรวจสอบเกียรติบัตร

เจ้าหน้าที่สามารถตรวจสอบความถูกต้องของเกียรติบัตรที่ออกโดยระบบ ได้โดยทำการกรอก Verification Code

ตรวจสอบเกียรติบัตร

ระบบจะทำการแสดงข้อมูลเกียรติบัตรหาก Verification Code ตรงกับข้อมูลที่มีในระบบ

ระบบรายงานปัญหา (เจ้าหน้าที่)

คู่มือการใช้งานระบบรายงานปัญหาสำหรับเจ้าหน้าที่

1

เพิ่มเพื่อน Line Official Account

เจ้าหน้าที่สามารถรายงานปัญหาที่พบในระบบ แจ้งข้อผิดพลาดต่างๆ และส่งข้อมูลปัญหาเข้าสู่ระบบ

รายงานปัญหา

เจ้าหน้าที่สามารถรายงานปัญหาที่พบในระบบ แจ้งข้อผิดพลาดต่างๆ และส่งข้อมูลปัญหาเข้าสู่ระบบ

2

ติดตามสถานะ

ติดตามสถานะการแก้ไขปัญหาที่รายงานไปแล้ว ดูความคืบหน้าในการแก้ไข และรับการแจ้งเตือนเมื่อมีการอัปเดต

ระบบสะสมแต้ม (ผู้ดูแลระบบ)

คู่มือการจัดการระบบสะสมแต้มสำหรับผู้ดูแลระบบ

1

จัดการระบบสะสมแต้ม

ภาพหน้าจอจัดการระบบสะสมแต้ม

ผู้ดูแลระบบสามารถจัดการระบบสะสมแต้มทั้งหมด ตั้งค่าการทำงานของระบบ จัดการบัญชีผู้ใช้ และควบคุมการเข้าถึงระบบ

2

ตั้งค่าระบบแต้ม

ภาพหน้าจอตั้งค่าระบบแต้ม

ตั้งค่าการทำงานของระบบแต้มสะสม กำหนดเงื่อนไขการให้แต้ม และปรับแต่งกฎการใช้งานแต้มสะสม

3

รายงานสถิติแต้ม

ภาพหน้าจอรายงานสถิติแต้ม

ดูรายงานสถิติการใช้งานระบบแต้มสะสม วิเคราะห์ข้อมูลการสะสมแต้ม และสร้างรายงานสำหรับผู้บริหาร

4

จัดการผู้ใช้งาน

ภาพหน้าจอจัดการผู้ใช้งาน

จัดการบัญชีผู้ใช้งานในระบบ เพิ่มผู้ใช้ใหม่ แก้ไขข้อมูล และจัดการสิทธิ์การเข้าถึง

ระบบจองรอบฐานกิจกรรม (ผู้ดูแลระบบ)

คู่มือการจัดการระบบจองรอบฐานกิจกรรมสำหรับผู้ดูแลระบบ

1

จัดการรอบฐานกิจกรรม

ภาพหน้าจอจัดการรอบฐานกิจกรรม

ผู้ดูแลระบบสามารถจัดการรอบฐานกิจกรรมทั้งหมด สร้างกิจกรรมใหม่ แก้ไขข้อมูล และจัดการตารางกิจกรรม

2

ตั้งค่าระบบจอง

ภาพหน้าจอตั้งค่าระบบจอง

ตั้งค่าระบบการจอง กำหนดเงื่อนไขการจอง เวลาเปิด-ปิดการจอง และข้อกำหนดต่างๆ

3

รายงานสถิติการจอง

ภาพหน้าจอรายงานสถิติการจอง

ดูรายงานสถิติการจองต่างๆ วิเคราะห์ข้อมูลการใช้งาน และสร้างรายงานประจำงวด

4

จัดการสิทธิ์การจอง

ภาพหน้าจอจัดการสิทธิ์การจอง

จัดการสิทธิ์การจองของผู้ใช้แต่ละคน กำหนดข้อจำกัดการจอง และจัดการกลุ่มผู้ใช้
